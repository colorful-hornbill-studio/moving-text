## Getting Started

### Prerequisites

1. JRE 8
2. JDK 8
3. NetBeans IDE 8

### Installation

1. Extract file ```moving-text-master.zip``` to your NetBeans project folder.
2. Rename folder ```moving-text-master``` to ```moving-text```.

### Usage

1. Open project ```moving-text``` in NetBeans IDE.
